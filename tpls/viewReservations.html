Reservation System
The Reservation app is an interactive visualisation that display information about system reservations from data stored in a SharePoint list. The user can quickly discover when conflicts occur with the reservations and take immediate action. The tool has many features that help speed up the process of discovering conflicts and managing the vast number of reservations. There are currently around 3,300 activities in the system.
